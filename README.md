# CasiBonus

Добро пожаловать в CasiBonus - сайт с промокодами для онлайн казино!

## О проекте

CasiBonus - это современный веб-сайт, предоставляющий информацию об онлайн казино и эксклюзивные промокоды для них. Проект создан с использованием передовых технологий веб-разработки.

## Особенности

- Список онлайн казино с описаниями
- Детальные страницы для каждого казино
- Информация о промокодах и бонусах
- Возможность копировать промокоды
- Адаптивный дизайн для мобильных устройств
- Темная/светлая тема
- Фильтрация по категориям игр (Crash, Roulette, Slots и другие)
- Отображение категорий игр для каждого казино

## Технологии

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <your-repo-url>
cd CasiBonus
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение в режиме разработки:
```bash
npm run dev
```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Сборка для продакшена

```bash
npm run build
```

Затем запустите продакшен-сервер:
```bash
npm run start
```
